# Inventory Optimization & Working Capital Intelligence

**Portfolio Project by Akshada Karade**  
MS Engineering Management · UMass Amherst · Power BI Certified · SAFe Agile

## About
An end-to-end supply chain analytics dashboard analyzing 2,674 automotive spare parts across 51 months of real demand history (Hyndman Car Parts Dataset).

## Analytical Modules
1. ABC/XYZ Classification (9-box segmentation)
2. Wilson EOQ with safety stock and reorder point
3. Newsvendor model for optimal order quantities
4. Working capital and days of supply analysis
5. Croston/SBA intermittent demand forecasting with MASE evaluation
6. Six Sigma layer: Cpk, SPC control charts, ANOVA
7. DMAIC framework with root cause analysis

## Tech Stack
Python · Plotly Dash · Pandas · NumPy · SciPy · Deployed on Render
